# Design References

## Primary reference: aniketj.dev
URL: https://www.aniketj.dev/

## Target style
- Ultra-dark background (#0a0a0f — #0f172a range)
- Accent: violet/purple gradient for name highlight
- Green badge for role/status
- Fonts: Inter (body) + JetBrains Mono (mono/code elements)
- Subtle glow effects (blur-3xl, primary/10 gradients)
- Gradient dividers between sections (from-transparent via-border to-transparent)
- Minimalist, technical, professional mood

## Hero section (Hero.png)
- Top: small green status badge ("Software Engineer")
- Large name: first name white, last name — purple/violet gradient
- Description paragraph with bold highlighted keywords
- Two CTA buttons: primary filled + ghost/outline with icon
- Stats row: numbers with labels (years, certs, projects)
- Right side: dark card with architecture/node diagram, glowing connection dots
- Subtle background glow behind elements

## Site sections (aniketj.dev)
1. Hero — intro + stats + visual
2. About — CLI-style profile card
3. Skills — expertise areas grid
4. Portfolio — featured project cards
5. Tech Stack — categorized tech badges
6. Contact — CTA with social links

## Visual effects
- Glow: absolute positioned blurs (blur-3xl) with primary color at low opacity
- Gradients: text-gradient on headings, bg gradients (primary/10 to transparent)
- Borders: very subtle (border/20, border-white/5)
- Animations: Framer Motion — fade-in, slide-up on scroll
- Hover: scale, glow increase, border brightness
