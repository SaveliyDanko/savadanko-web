05 — Shadows & glow

07 — Voice

Senior engineer, calibration interview.

Precise, evidence-led, light. Numbers when you have them. Active voice. No emoji rocket ships.

Do
"Backend engineer. I build distributed systems in Java and Kotlin and care about the parts that don't break at 3am."
"Cut p99 latency from 340ms to 80ms by switching to a streaming model."
"Built X to solve Y. Result: Z."
Don't
"🚀 Passionate full-stack ninja crafting amazing experiences with cutting-edge tech."
"Leveraging synergistic blockchain solutions to disrupt the paradigm."
"Coding rockstar 🎸 building the future ✨"
08 — Tokens

Drop-in for any new project.

Three artefacts ship with this brand: tokens.json, tokens.css, and a Tailwind v4 @theme block. Pick the one that matches your stack and start.

tokens.css · CSS variables
/* Drop into any project, then: */
.button {
  background: var(--brand-primary);
  color: white;
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-primary-btn);
}
Tailwind v4 · @theme block
@theme {
  --color-primary: #8b5cf6;
  --color-section-label: #6f87ff;
  --color-surface: #010105;
  --color-heading: #ffffff;
  --font-sans: "Inter";
  /* …see tokens.css for full block */
}
