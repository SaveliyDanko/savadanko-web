# Savely Danko Portfolio

Personal portfolio built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Deployment

The repository is prepared for deployment on a custom domain.

See the full guide here:

- `docs/deploy-guide.md`
- `.env.production.example`

## Notes

The app uses client-side routing with production fallbacks for common static hosts.
